Politique de Confidentialité - GEREKOM

Dernière mise à jour : 3 mars 2026
🔒 POINT IMPORTANT : GEREKOM est une application 100% côté client. AUCUNE donnée n'est collectée, stockée ou transmise sur nos serveurs. Toutes vos données de stock (produits, ventes, achats) restent uniquement sur votre ordinateur.

1. Qui sommes-nous ?

GEREKOM est une application de gestion de stock accessible à l'adresse [gerekom.github.io]. Notre application fonctionne entièrement dans votre navigateur, sans serveur backend.

2. Quelles données sont stockées et où ?

📌 RÉPONSE SIMPLE : Nous ne collectons RIEN. Tout reste chez vous.

2.1 Données de l'application (localStorage)

Les données suivantes sont stockées UNIQUEMENT sur votre navigateur (votre ordinateur ou téléphone) :

  • Produits : Les noms, prix, quantités que vous saisissez
  • Ventes : L'historique des ventes que vous enregistrez
  • Achats : L'historique des achats que vous enregistrez
  • Noms des clients (optionnel) : Si vous saisissez un nom dans le champ "Client"
  • Noms des fournisseurs (optionnel) : Si vous saisissez un fournisseur

Ces données ne quittent JAMAIS votre appareil. Nous n'y avons pas accès. Vous pouvez les supprimer à tout moment via le bouton "Formater les données".

2.2 Données de session (sessionStorage)

  • from_landing : Un simple "drapeau" technique pour savoir si vous venez de la page d'accueil
  • S'efface automatiquement quand vous fermez le navigateur

3. Cookies et publicité

3.1 Cookies Google Ads (si vous en voyez)

La page d'accueil (landing page) peut afficher des publicités Google Ads. Google utilise ses propres cookies pour :

  • Afficher des publicités pertinentes
  • Limiter le nombre de fois que vous voyez une même annonce
  • Mesurer l'efficacité des campagnes

Ces cookies sont gérés par Google, pas par nous. Pour en savoir plus : Politique publicitaire de Google

3.2 Google Analytics (statistiques)

Nous utilisons peut-être Google Analytics pour savoir combien de personnes visitent la page d'accueil. Google Analytics utilise des cookies pour collecter des informations anonymes :

  • Pages visitées
  • Temps passé sur le site
  • Type d'appareil (mobile/ordinateur)
  • Pays approximatif

5. Partage des données

Nous ne vendons, ne louons et ne partageons aucune de vos données personnelles. Comme expliqué, nous n'y avons même pas accès !

Les seules données partagées le sont via les cookies Google (Ads et Analytics) et sont régies par la politique de confidentialité de Google.

6. Vos droits

Même si nous ne collectons rien, vous avez des droits très simples à exercer :

  • Supprimer vos données : Utilisez le bouton "Formater les données" dans l'application
  • Exporter vos données : Utilisez le bouton "Sauvegarder" pour télécharger un fichier
  • Refuser les cookies : Via les paramètres de votre navigateur
  • Contacter Google : Pour les données Ads/Analytics, contactez directement Google
